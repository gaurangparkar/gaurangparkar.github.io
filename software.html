Software

I use MacOS as my operating system since mid 2022. I had used Arch Linux and Ubuntu for many years before that but switched to MacOS due to the performance and power efficiency of Apple Silicon chips.

I use neovim as my main editor for most applications like building this website. I use VS Code for larger projects.

I use the Julia programming language for most of my calculations. It has almost C/C++ like performance while being sufficiently high level and easy to use.

I also use Wolfram Mathematica for small calculations and some libraries in C and C++. I can use python, however I have not used it much since I started using Julia as I find it to be quite slow.

I use Latex for writing documents, and beamer to make presentations. I also like the MacOS presentation app called Keynote.
