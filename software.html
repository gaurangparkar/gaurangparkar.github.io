Software

Operating system

I use MacOS as my operating system since mid 2022. I had used Arch Linux and Ubuntu for many years before that but switched to MacOS due to the performance and power efficiency of Apple Silicon chips. Here are some useful tools to make MacOS more usable. Most of these would require you to have brew and macports.

  • Rectangle : Rectangle is a window manager for MacOS with useful keyboard shortcuts. When using MacOS for the first time I was frustrated due to the lack of a native window manager.
  • Neardrop : Brings the functionality of AirDrop on an android phone using Quickshare/Nearby share.
  • MonitorControl : Controls the brightness of external displays.
  • Skim : A better pdf reader than preview.

Programming languages

  • I use neovim as my main editor for most applications like building this website. I use VS Code for larger projects.
  • I use the Julia programming language for most of my calculations. It has almost C/C++ like performance while being sufficiently high level and easy to use.
  • I also use Wolfram Mathematica for small calculations or when symbolic manipulation in needed.
  • For high performance computing I use libraries in C and C++ especially because a lot of code is already available for use.
  • I started learning programming with python for a computational physics class, however I have not used it much since I started using Julia as I find it to be quite slow and clunky.

Writing reports

  • I use Latex for writing documents.
  • I mainly use beamer to make presentations. I also like the MacOS presentation app called Keynote.

Useful Libraries

These are some libraries that I have used for my work over the past few years, in no particular order of importance.

  • OpenQCD-fastsum : This C package written and maintained by the fastsum collaboration extends the OpenQCD code for anisotropic lattices.
  • SIMULATeQCD : This C++ package is designed to run Lattice QCD code on Multiple GPU’s. I used this package to do gauge fixing and measure Wilson Line correlators on already generated configurations.
  • TensorOperations : A julia package for fast and efficient tensor operations using the Einstein index notation.
  • flux : This is a library for machine learning in native Julia.
  • scikit-learn : Machine learning library for python.
  • diffgeo : A Mathematica package for doing GR-type tensor algebra and calculus.
  • BR Method : A C package that implements the Bayesian Reconstruction(BR) method for extracting spectral function from Lattice QCD correlators.
